## Installation

```bash
git clone https://github.com/n0tNoah/django-cookiecutter-rest
pip install -r requirements.txt
python3 manage.py migrate

#optional to start contributing
pip install pre-commit
pre-commit install

```

## running locally

```bash
python3 manage.py runserver
```
